Installation Instructions for MultiMC

  1. Download modpack
  2. Open MultiMC. Click add instance
  3. Select Import Modpack and click the ... button.
  4. Find the modpack zip you downloaded and click open.
  5. Optional Give the pack a descriptive name, assign it to a group, or change the icon
  6. Click ok

Server information

Hostname

modded.wolff.io (this site!)

Specs

Running on a Google Cloud preemptible n1-standard instance

  • 1 vCPU
  • 3.75GB RAM
  • Location: Iowa

The server is configured to run from 5pm-2am weekdays, and 10am-2am weekends. Running continuously would be a waste of money.

Server is currently ...

Since the server runs on a preemptible VM, Google may decide they need the resources and shut down the machine.
