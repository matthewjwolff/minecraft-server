Information about Minecraft Servers

Vanilla 1.14.2

Installation Instructions for MultiMC

  1. Open MultiMC. Click add instance
  2. Select Vanilla 1.14.1 from the list.
  3. Optional Give the pack a descriptive name, assign it to a group, or change the icon
  4. Click ok
